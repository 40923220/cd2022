w2(03.03)

建立倉儲

1.到cmstemplate可攜系統下載portablekmol_base.7z、 Python395.7z、portablegit_2.31.1.7z後解壓縮，將解壓縮好的 Python395.7z、portablegit_2.31.1.7z丟入portablekmol_base.7z解 壓縮好的data資料夾。

2.登入Github到老師網頁的About點選網址進入後按Use this template,輸入倉儲名稱後按Create repository from template。

3 進自己的倉儲後，按上方的Settings->Pages->None改main->Save->生 成網頁網址。

4 開下載好的ipv6，將 ipv4拉進白窗，將 Python38都改成 Python395、 portablegit 改成 portablegit2.31.1，最下面將改成這樣。

5 開小黑窗

git clone --recurse-submodules https://...新建的倉儲網址.git， 跑完之後會發現data->tmp->多了倉儲名稱的資料夾。

6 小黑窗輸稱 

  cms=>複製網址進入編輯的地方，預設登入密碼為“admin ”

7 回到倉儲建立Tokenremote下面的url=https://github.com...在//和 github 中間貼上剛剛複製的token和@

8.完成上述即可

CoppeliaSim 操控流程

1.將繪製好並組合好的檔案轉成STL檔

2.開啟Coppeliasim匯入剛轉好的STL檔

3.將模型移到適當位置

4.分解模型

5.將多餘的零件關閉所有圖層的顯示(螺釘、螺帽...)

6.把所有模型簡化

7.加上程式碼

8.裝上馬達並將馬達取對應程式碼的名字

9.開始連接模型以及馬達

10.模擬作動

心得:經過這次W8的作業後,在這次之後了解coppeliasim版本差異，語法功能不同，舊版不能開新版檔案等差異。coppeliasim的檔名不使用中文、檔案放置位置的資料夾也都使用英文，不然在模擬時容易出現找不到檔案。
